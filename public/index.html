Project Status

PS2Alerts is still being written, below is a summary of where we're at for each module:

Websocket

65%

The websocket collects the real-time data from census, does logic calculations on it, then passes the info off to our API. It is the beating heart of the application and takes the longest to program.

API

10%

The API accepts the information from the websocket and stores it to our database. It also allows people to read the data via the website and special API access.

Website

What you're viewing now. It's not much, but it's honest work. Eventually this will show the full range of statistics and metrics in real time for each alert.
