23rd July 2017

Change Log notification badges!

Much wow. Much badges.

The Change Log button on the navbar (at the top) will now show the number of unread changes for you to discover.

While it may not seem like a huge change, it's more of a quality of life thing. Currently announcements to changes to the site end up being on the homepage, which takes up about a half of the screen real-estate upon opening the page. It's highly likely (and on mobiles extremely likely) that you'll miss the all important statistics.

Therefore it made more sense to me to remove all the cruft from the homepage and announce changes via a "badge" (iOS sheeple will know this well) which will glow red when there's a new update in the change log that you've not read.

I've also done some various tidying up of the navbar in general. I'm going to also take a look at how the mobile navbar is set up as I'm not 100% happy with it.

19th June 2017

Combat Statistics Update

Having been absent from the website for quite some time now (and was on previous versions), I've added the Combat Statistics section, available on the home page under COMBAT.

This section has two subsections - Overall Combat and Class Combat statistics.

Overall Combat Section

This section shows the overall combat statistics (kills, deaths etc) and displays not only the total across all alerts also but the per server, per faction distribution. This way you're able to see exactly how many kills the VS got on Miller across all alerts.

The statistics that are shown are for Kills, Deaths, Teamkills, Suicides and Headshots.

Class Combat Section

The section shows the comparisons across classes on a per metric, per faction basis e.g. Kills across all classes. These are further broken down into faction compositions, so we can see exactly how many suicides a TR Engineer has made (and there's loads!)

I've also added a KDR graph for convenience. This allows people to see relative KDRs across each class. As of writing, the MAX reigns supreme.

Find all the changes above on the Homepage, link below.

19th June 2016

Homepage Update & Project Statuses consolidated

Victory Timeline

This handy graph shows the victories for each faction over time, allowing you to see the trends that are occurring. This graph will be improved over time, with labels detailing when major patches landed so you can see their affect on alerts coming soon.

Victory Breakdowns

There's now 4 new statistics panels which show you how each server's victories are distributed. E.g. You're now able to see which servers the Vanu Sovereignty are doing best on Indar. (As of writing, that's Miller and Cobalt!).

Global Server Victories - Victories broken down by each server, showing how each faction are doing on each server.

Global Continent Victories - Victories broken down by continent, showing which faction is dominating Indar! (As of writing, that's the TR!)

Per-Server Continental Victories - Victories broken down by Server and by Continent, which allows you see how each continent fairs for each faction on each server.

Faction Victory Contributions - Shows a breakdown of the total number of wins, per faction, and how they contribute to the overall faction percentages. Shown in both a segmented bar chart and in tabular form. Interested to see which servers are pulling their weight? Now you can find out!

29th April 2016

Outfit Profiles

Outfit profiles are now live! Wonder how you compare to the rest of your outfit mates? Now you can find out!

The outfit profiles show stats such as:

  • Kills, Deaths, Teamkills, Suicides and headshots across all alerts recorded
  • Averages, such as K/D, Kills & Deaths per alert etc
  • Facility Statistics, along with all counts of captures and defences for every facility for that outfit
  • A list of all alerts that the outfit has participated in

Player Profiles

I'm happy to present the first iteration of Player Profiles! With player profiles, you'll be able to see the statistics for any particular player we have on record, including stats such as:
